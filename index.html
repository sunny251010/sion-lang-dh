Trình mở nhanh buổi thờ phượng

Nhấn một lần để hiện 3 buổi Sáng, Chiều, Tối. Khi chọn một buổi, trang sẽ mở tab mặc định worship.watv.org và toàn bộ tab bài ca mới tương ứng.

Trang sẽ tự tải dữ liệu từ API này khi mở website. Nếu API lỗi hoặc success là false, web sẽ quay về dữ liệu cứng hiện tại.
